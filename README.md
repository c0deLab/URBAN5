Based on URBAN5 by Nicholas Negroponte

To setup:

yarn install

To run:

yarn start

Key controls:
v: 2D slice view
b: 3D walk through
x: north slice view
a: east slice view
w: south slice view
d: west slice view
s: top plan view
c: bottom plan view
1: add cube
2: add tree
3: add roof left
4: add roof right
5: remove element
↑: move forward in 3D walk through
↓: move backward in 3D walk through
